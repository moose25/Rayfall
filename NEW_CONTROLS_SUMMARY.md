# Rayfall - New Control Panel Features

## Control Panel Layout (March 13, 2026 Update)

### Row 1: Date Range & Primary Controls
- **Start Date** picker
- **End Date** picker
- **Load QSOs** button
- **Show Lines** checkbox - Display lines from QTH to QSOs
- **Color by Band** checkbox - Color markers by ham radio band
- **Basemap Selector** dropdown - Choose map style (Light/Dark/Topo/Satellite/Streets)

### Row 1 (Continued): NEW Grid & Display Controls
- **Show Grid Squares** checkbox - Toggle Maidenhead grid overlay
- **Grid Precision** selector - Choose between 4-char or 6-char grids
- **Show Pins** checkbox ⭐ NEW - Hide/show all QSO location markers
- **Show QTH** checkbox ⭐ NEW - Hide/show your operating station (⚡)
- **Export Map (300 DPI)** button - Download high-res print-ready JPEG

### Row 2: Filter Controls
- **QSOs shown** counter - Live count of displayed contacts
- **Bands** filter - Checkboxes for each band with color indicators
- **Modes** filter - Checkboxes for each mode

## Use Cases

### Scenario 1: Clean Grid Square Visualization
Perfect for creating a "worked grids" map without clutter:
1. Load your QSOs
2. Check "Show Grid Squares"
3. Select "4-char grids" for broader coverage view
4. Uncheck "Show Pins" to hide individual contacts
5. Uncheck "Show QTH" for maximum focus on worked grids
6. Export to 300 DPI for printing

### Scenario 2: Detailed Contact Map
Show everything for analysis:
1. Load your QSOs
2. Keep "Show Pins" checked
3. Check "Show Lines" to see propagation paths
4. Keep "Show QTH" checked to see your location
5. Use band/mode filters to analyze specific aspects

### Scenario 3: Grid Square Contest Planning
See which grids you've worked with detail:
1. Load contest period QSOs
2. Check "Show Grid Squares"
3. Select "6-char grids" for maximum detail
4. Use "Show Pins" to see exact QSO locations within grids
5. Filter by band to see grid coverage per band

### Scenario 4: Print-Ready Award Maps
Create clean maps for award applications:
1. Configure display exactly as desired
2. Hide elements you don't want in the print
3. Select appropriate grid precision
4. Click "Export Map (300 DPI)"
5. Print at full quality

## Technical Notes

- All toggles update instantly without reloading data
- Grid precision changes automatically redraw overlays
- Export captures current visible state
- All settings are preserved during filtering operations
- QSO data remains available even when pins are hidden

## Keyboard-Free Operation

All controls can be accessed via mouse/touch:
- ✅ No keyboard shortcuts required

- ✅ All selections are dropdowns
- ✅ Single-click operations
