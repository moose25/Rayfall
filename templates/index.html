📖 How to Use Rayfall

🔑 Method 1: Using QRZ API

  1. Click the "Change API Key" button (top-right)
  2. Enter your QRZ Logbook API key
  3. Select a Start Date and End Date
  4. Click "Load from QRZ"
  5. Your contacts will appear on the map! 🗺️

Getting your API Key: Visit logbook.qrz.com → Settings → API

📁 Method 2: Import ADI/ADIF Files

  1. Export your log as an ADI file from your logging software:
    • QRZ Logbook: Logbook → ADIF → Download
    • Log4OM: File → Export → ADIF
    • WSJT-X: File → Export ADIF
    • N1MM Logger+: File → Export → ADIF
  2. Click "Import ADI File" button
  3. Select your .adi or .adif file
  4. Your contacts load instantly - no API key needed! ⚡

Benefit: Works offline, completely private, no internet required!

🎨 Customizing Your Map

Click the "⚙️ Display Options" button to access:

  • Map Style: Choose Light, Dark, Topographic, Satellite, or Streets
  • QTH Station Icon: Clean shapes (Dot, Plus, Square, Signal) or emoji (⚡ 📍 🏠 📡 🗼 ⭐), plus a color picker and an optional grid label under each marker
  • Pin Icons: Change marker style (Teardrop, Circle, Square, Star)
  • Grid Squares:
    • Toggle grid square overlay on/off
    • Choose 4-character or 6-character precision
    • Pick custom colors with the color picker
    • Adjust opacity (0-100%)

📡 Multiple QTHs

Operating from more than one location in a single log — travel, POTA, Field Day? Rayfall reads MY_GRIDSQUARE / MY_LAT/MY_LON from each QSO and groups them:

  • Every unique QTH gets its own marker on the map (grouped at 6-character grid precision).
  • Each contact popup shows From QTH: so you always know where you were when you made the QSO.
  • Toggle Color Pins by QTH to color every contact by its originating station.
  • Use the QTHs filter row to isolate contacts from one trip stop.

🔍 Filtering Contacts

  • Show Lines: Toggle lines from each QTH to its contacts
  • Color by Band: Color-code pins by frequency band
  • Color by QTH: Color-code pins by originating station (wins if both are on)
  • Band / Mode / QTH Filters: Check/uncheck to hide contacts by band, mode, or originating QTH

The QSO counter updates live as you filter!

🖨️ Exporting for Printing

  1. Arrange your map exactly how you want it
  2. Use Display Options to customize appearance
  3. Click "Export Map (300 DPI)"
  4. A high-resolution JPEG will download (3300×2550 pixels)
  5. Perfect for printing at 11"×8.5"!

Tip: Hide pins and show only grid squares for clean award maps!

💡 Pro Tips

  • 🎯 Contest Analysis: Import contest logs and visualize propagation patterns
  • 📊 Award Progress: Use 6-char grids to see exactly what you've worked
  • 🌈 Custom Themes: Match grid colors to your award theme
  • 💾 Multiple Logs: Import different ADI files for comparison
  • 🔒 Privacy: ADI import means your logs never leave your computer
